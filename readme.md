## 可用命令
在此项目目录中，可以运行一下命令：
### `npm run dev:client`
在开发模式下运行前端页面。<br />
打开[http://localhost:8081](http://localhost:8081)在浏览器查看。
当文件发生修改时，页面会自动重新加载。

### `npm run watch:node`
在开发模式下运行服务端项目。
当文件发生修改时，服务会自动重新启动。
### `npm run build`
生成生产环境App到`out`文件夹。
